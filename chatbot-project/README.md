# 🤖 React ChatBot

A simple rule-based chatbot built with **React**, **HTML**, and **CSS** that responds to a predefined set of user queries. The project demonstrates the practical use of React Hooks to create an interactive and responsive chat interface.

## Features

* Interactive chat interface with user and bot messages.
* Welcome message displayed on initial load.
* Welcome message disappears after the first user interaction.
* Input field and send button for seamless communication.
* Automatic scrolling to the latest message.
* Instant responses to predefined questions.

## Technologies

* React
* HTML5
* CSS3

## React Hooks Used

* **useState** – Manages chat messages, user input.
* **useEffect** – Triggers auto-scrolling whenever new messages are added.
* **useRef** – References the latest message element to implement smooth auto-scroll.

## Purpose

This project was developed to strengthen my understanding of React fundamentals, including component-based architecture, state management, conditional rendering, and DOM manipulation using React Hooks.
